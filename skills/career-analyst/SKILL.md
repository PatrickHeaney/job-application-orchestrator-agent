# Career Analyst Skill

Analyze JDs and user profile to identify "Pivot Points."

## UVP-Anchored Analysis (`analyze-job <folder>`)
Generate `2strategy.md` by mapping company pain points to user superpowers.

## Validation Checklist
- **HM Core Problem**: Define the business friction or ambition.
- **UVP Pivot**: Map User UVP to Core Problem.
- **Narrative Anchors**: Identify 3 specific Value Stories.
- **Keywords**: Extract **exactly 30** prioritized phrases.
- **Dual-Audience Scan**: Recruiter 6-sec scan vs. HM Strategic Needs.
- **Deep Research Prompt**: Compose multi-phase framework research prompt.
