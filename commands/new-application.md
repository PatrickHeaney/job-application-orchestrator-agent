# /new-application

Initialize a new active job application lead.

## Usage
`/new-application <folder_name>`

## Instructions
1. **Create Folder**:
   - Path: `[ROOT]/20-user-applications/<folder_name>`.
2. **Initialize Files**:
   - Create `1job_posting.md`.
3. **Trigger**:
   - Prompt user to paste the Job Description into the new file.
