# /set-root

Establish the Sovereignty Boundary for the orchestration session.

## Usage
`/set-root`

## Instructions
1. **Interactive Path Selection**:
   - Ask: "Where is the root of your Career Orchestration workspace?"
   - Verify path exists.
2. **Boundary Lockdown**:
   - Set path as session **[ROOT]**.
   - Ignore all folders outside [ROOT] unless explicitly referenced.
3. **Verify Scaffolding**:
   - Check for: `20-user-applications/`, `30-archived-user-applications/`, `10-user-profile/user_profile.md`.
